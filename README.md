
Split300 — это приложение для совместного учёта общих расходов (квиттинга). Оно помогает компаниям друзей, семьям, парам или соседям легко и прозрачно фиксировать, кто кому и сколько должен.

Ключевые возможности:
Учёт долгов: Внесение общих трат (еда, аренда, путешествия, коммуналка).
Автоматические расчёты: Приложение само вычисляет, кто, кому и сколько должен, чтобы минимизировать количество переводов.
Балансы в реальном времени: Все участники группы видят текущие балансы и долги.
Главная цель: Упростить учёт общих финансов и избежать неловких ситуаций с напоминаниями о деньгах, сохраняя дружеские отношения.
